О приложении

Shop Notes

Progressive Web App для управления заметками

Версия 1.0.0

Архитектура: App Shell с HTTPS

Возможности

  • Работает в режиме offline (полностью оффлайн приложение)
  • Защита по HTTPS (безопасное соединение)
  • Архитектура App Shell (мгновенная загрузка интерфейса)
  • Синхронизация данных в localStorage
  • Адаптивный дизайн (работает на всех устройствах)
  • Возможность установки как приложение

Технологии

  • HTML5
  • CSS3 (современные стили и анимации)
  • JavaScript (ванильный, без фреймворков)
  • Service Worker API
  • Web App Manifest
  • LocalStorage API
