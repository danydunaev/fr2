# Интернет-магазин (Frontend + Backend)

Учебный проект интернет-магазина, выполненный в рамках практической работы №4 по дисциплине "Фронтенд и бэкенд разработка". Проект состоит из двух частей:

- **Сервер** на Node.js + Express, предоставляющий REST API для управления товарами (CRUD).
- **Клиент** на React, взаимодействующий с API через axios.

## Технологии

### Backend
- Node.js
- Express
- nanoid (генерация уникальных ID)
- cors (для разрешения кросс-доменных запросов)

### Frontend
- React (Create React App)
- Axios (HTTP-клиент)
- Sass (стилизация)
- React Hooks (useState, useEffect)

## Требования

- Node.js (версия 14 или выше)
- npm (или yarn)
